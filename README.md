### Requirements:
- Modern browser capable of running JavaScript
- Modern browser capable of webGL2
---
### JavaScript Ray Tracer
#### Above and beyond spec:
- Control over canvas size
- Control over sphere transform
- Control over scale
- Control over colour of sphere
- Togglable Gamma Correction
- Control over amount of multisamples
- Control over light direction
- Control over shadow opacity
- Control over specular intensity
- WebGL (See Below)

### WebGL
#### Feature Set:
- Rotating cube
- Normal based lighting (diffuse, ambient)
- Switch between textures and colours
- Pause/Play Button
- Change face colour
- Control over rotation speed
